## Usage

Install package:

```sh
# npm
npm install @unicli/core

# yarn
yarn add @unicli/core

# pnpm
pnpm install @unicli/core
```

Import:

```js
// ESM
import { } from '@unicli/core'

// CommonJS
const { } = require('@unicli/core')
```
